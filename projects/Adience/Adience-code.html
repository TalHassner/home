Code

This page provides links to the following code and software
  • New! MATLAB face frontalization code
  • 2D, in-plane alignment of faces

Face frontalization

MATLAB code for synthesizing front-facing views of faces from unconstrained photos is available from the frontalization project page. Please see that page for download and citation instructions.


In-plane alignment of faces


Code used to align face photos, used in the paper Age and Gender Estimation of Unfiltered Faces (See our publications page for more information). The code is built as an extension of the method originally described by Zhu and Ramanan, and uses their code.

Please see our paper for a detailed description of the alignment method.

Download: https://github.com/eranid/adience_align
Citation: If you use this code, please cite the paper Eran Eidinger, Roee Enbar, and Tal Hassner, Age and Gender Estimation of Unfiltered Faces, (See our publications page for more information).
